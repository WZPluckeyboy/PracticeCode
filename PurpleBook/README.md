# 算法竞赛入门(紫书)

> 这里是我读紫书过程中的一些代码和笔记。

## 第一章

### note01

> 1. 输入前不要打印提示信息，输入完毕后立即终止程序，不要等待用户按键。
> 2. 算法竞赛中不要使用头文件conio.h，包括getch()、clrscr()等函数。
> 3. 每行输出均以回车符结束，包括最后一行，除非特别说明，每行行首不应有空格，但行末通常可以有多余空格。输出的每两个数字或者字符串之间用单个空格隔开。
