# 2、8、10、16进制间的相互转换

# 使用方法

输入：

退出程序：`end`或`END`

二进制：`b`或`B`

八进制：`o`或`O`

十进制：`d`或`D`

十六进制：`h`或`H`

# 程序介绍

以字符串的方式获取输入，以字符串的形式输出。

将输出的字符串转换为十进制数，再由十进制数转换为其他进制的字符串。转换成字符串的过程用到栈。
